Bones

Bones is a front end web development tool. I started making it when I started learning Gulp and the command line and have been adding to it since then.

Using Sass I have also built a 12 column grid system for quicker page layouts and some useful functions, placeholders and helper classes.

It includes autoprefixer, sass and css minification. As well as concatinating Javascript files and running gulp-modernizr to automatically create custom Modernizr files.
